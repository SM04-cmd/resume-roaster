# 🔥 AI Resume Roaster

A brutally honest AI-powered tool that roasts your resume 
and gives you actionable fixes to land your dream job.

## ✨ Features
- Paste your resume → get instant AI feedback
- Brutal roast + specific fixes
- Score out of 10
- No fluff, no sugarcoating

- Vercel

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
